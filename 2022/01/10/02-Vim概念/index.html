Vim 概念

一 模式

模式 名称 说明
normal mode 普通模式（正常模式） 打开 Vim 后进入的默认模式
insert mode 插入模式 普通模式下按下 i、I、a、A、o、 O 等进入
insert-normal mode 插入-普通模式 在插入模式下使用 ctrl-o 进入，执行一次普通模式命令后返回插入模式
command-line mode 命令行模式 普通模式下按下 : 键进入，命令行模式中执行的命令又被称做 Ex 命令
Visual Mode 可视化模式 可视化模式下可以选择一块编辑区域，然后对选中的内容执行各种操作
Operator-Pending mode 操作符待决模式 命令通常为 operator motion，当只键入了操作符 operator 还没有键入动作 motion 时即为操作符待决模式

二 缓冲区

2.1 概述

  1. Vim 打开文件进行编辑时其实编辑的是文件在内存中的映像（即Vim 中的缓冲区）
  2. 当使用 Vim 对缓冲区进行操作时，磁盘文件内容与内存映像不再一致，执行 :w、:x 等命令时便是将缓冲区的内容回写到文件
  3. 每打开一个文件，Vim 就会创建一个对应缓冲区，缓冲区不仅包含了文件的内容，也记录了该文件的编辑状态等配置信息，包括激活、隐藏、上次编辑行数等
  4. 删除一个缓冲区并不会影响缓冲区所关联的磁盘文件，只是把该文件在内存中的映像删掉

1.2 缓冲区操作

1 查看

命令 说明
:ls 列出所有缓冲区
:buffers 列出所有缓冲区
结果 说明
%a 表示当前缓冲区
# 表示轮换缓冲区
+ 表示修改过但未保存的缓冲区，如果当前缓冲区未保存就切换，则当前缓冲区会被标记成隐藏缓冲区
h 表示隐藏缓冲区

2 切换

命令 简写 说明
:bnext :bn 切换到下一个缓冲区
:bprevious :bp 切换到上一个缓冲区
:bfirst :bf 切换到第一个缓冲区
:blast :bl 切换到最后一个缓冲区
:buffer n :b n 切换到编号为 n 的缓冲区
:buffer file :b file 切换到指定文件的缓冲区
ctrl-^ 在当前和上一个缓冲区间来回切换

3 删除

命令 简写 说明
:bdelete :bd 删除当前缓冲区:
:bdelete n1 n2 ... :bd n1 n2 ... 删除指定编号的缓冲区
:m,n bdelete :m,n bd 删除编号从 m 到 n 的缓冲区

三 屏幕和标签页

3.1 分屏

1 分隔窗口

命令 Ex 命令 说明
ctrl-w s :split 水平分屏当前文件（上下）
ctrl-w v :vsplit 垂直分屏当前文件（左右）
:sp 文件名 水平分屏并用另一个屏幕打开新文件
:vsp 文件名 垂直分屏并用另一个屏幕打开新文件
:new 水平新建窗口编辑一个新文件
:vertical new 垂直新建窗口编辑一个新文件

2 切换窗口

命令 说明
ctrl-w w 在窗口间切换
ctrl-w h 切换到左侧窗口
ctrl-w j 切换到下侧窗口
ctrl-w k 切换到上侧窗口
ctrl-w l 切换到右侧窗口
ctrl-w H 把当前窗口移到最左边
ctrl-w J 把当前窗口移到最下边
ctrl-w K 把当前窗口移到最上边
ctrl-w L 把当前窗口移到最右边

3 关闭窗口

命令 Ex 命令 说明
ctrl-w q :close 关闭当前活动窗口
ctrl-w c :close 关闭当前活动窗口
ctrl-w o :only 只保留当前活动窗口，关闭其它窗口
:qall 关闭所有窗口
:wall 保存所有窗口
:wqall 保存并关闭所有窗口

4 窗口布局

命令 说明
ctrl-w = 所有窗口等宽、等高
ctrl-w _ 最大化活动窗口的高度
`ctrl-w `
ctrl-w + 增加活动窗口的高度
ctrl-w - 减少活动窗口的高度

3.2 标签页

1 概述

  1. Vim 标签页是可以容纳一系列 Vim 窗口的容器
  2. 一个标签页可能包含一个或多个窗口

2 查看标签页

命令 说明
:tabs 显示所有标签页，>：表示当前缓冲区，+：表示修改过的缓冲区

3 创建标签页

命令 说明
:tabedit 文件名 在新标签页中打开指定的文件
:tab new 文件名 在新标签页中打开新文件
:tab split 在新标签页中打开当前文件

4 切换标签页

命令 说明
:tabn 切换到下一个标签页
gt 切换到下一个标签页
:tabp 切换到上一个标签页
gT 切换到上一个标签页

5 关闭标签页

命令 说明
:tabc 关闭当前标签页及其中的所有窗口
:tabonly 关闭其它所有标签页，只保留当前标签页

6 移动标签页

命令 说明
:tabm 将当前标签页移动到最后位置
:tabm 3 将当前标签页移动到第 3 个标签页之后

四 寄存器

4.1 概述

  1. VIm 的删除、复制和粘贴需要使用寄存器，在命令前加 "寄存器名 前缀的方式可指定寄存器
  2. 寄存器被存储在 ~/.viminfo 中, 在下次重启 Vim 时仍会加载

4.2 寄存器种类

符号 名称 说明
" 无名寄存器 默认使用
0 复制专用寄存器 当使用 Vim 命令 y{motion}时，复制的文本不但被拷贝到 无名寄存器，而且也被拷贝到了 复制专用寄存器中
_ 黑洞寄存器 有去无回
* 当前选择区 表示正被选中的文本
+ 系统剪贴板 将内容复制到系统剪贴板供外部使用，或者从系统剪贴板获取外部复制的内容
a-zA-Z 有名寄存器 使用 a-z 引用有名寄存器为 覆盖，使用 A-Z 引用有名寄存器为 追加
1-9 数字寄存器
: 冒号寄存器 保存着最后执行的命令行命令

4.3 寄存器操作

命令 说明
"寄存器名称 指定使用的寄存器，缺省为无名寄存器，即 ""
:reg 查看寄存器内容
:reg 寄存器名称 查看指定寄存器内容

五 标记

5.1 概述

  1. 一般当你执行一个会将光标移动到本行之外的命令，该移动即被称为一个 “跳转”
  2. 跳转包括搜索命令 / ? n N
  3. 跳转不包括：
    • f t 等行内搜索命令
    • h j k l 等移动命令
    • w e 等单词移动命令
  4. 可以使用有名标记或内置的预定义标记记录跳转位置

5.2 命令

命令 说明
m标记名 用指定名称标记当前位置
`标记名 跳转到指定标记位置（反引号）
'标记名 跳转到标记所在行的行首（单引号）
:marks 查看所有标记
:marks 标记名 查看某个标记
:jumps 查看跳转列表
ctrl-o 跳转到一个较老的位置
ctrl-i 跳转到一个较新的位置

5.3 标记名

标记 设置者 说明
a-z 用户 仅对当前文件有效
A-Z 用户 全局有效
0-9 viminfo 0 代表 viminfo 最后一次被写入的位置。实际使用中，0 代表 Vim 进程最后一次结束的位置，1 代表倒数第二次，以此类推

5.4 预定义标记

标记 说明
. 最后一次修改文件的位置
^ 上一次插入的位置
' 跳转前的光标位置
" 最后编辑的光标位置
[ 最后修改的开始位置
] 最后修改的结束位置
< 上一次可视模式高亮选区的起始位置
> 上一次可视模式高亮选区的结束位置
( 当前句子的开头
) 当前句子的结尾
{ 当前段落的开头
} 当前段落的结尾

六 宏

6.1 概述

  1. Vim 中的宏可理解为将一系列操作缓存到某个寄存器中，需要的时候直接使用宏即可
  2. 宏的目的即重复使用

6.2 宏录制和使用

命令 说明
:q寄存器名 开始录制，按 q 结束录制；寄存器使用 a-zA-Z0-9
@寄存器名 执行宏
n@寄存器名 执行 n（正整数） 次宏
@@ 重新执行上次执行的宏

6.3 修改宏

  1. 首先将寄存器中的内容粘贴到文件中
  2. 修改内容
  3. 修改之后再复制到寄存器中覆盖即可达到修改宏的目的

6.4 保存宏

  1. 使用 Linux 的 let 命令可直接将内容存储到寄存器中
  2. let @寄存器名 =""
