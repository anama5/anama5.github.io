Vim 操作

一 概述

1.1 命令格式

操作符 说明
d 删除（delete）
y 复制（yank 抽出到寄存器）
c 修改（change）
p 粘贴（put）
> 增加缩进
< 减少缩进
= 自动缩进
g~ 反转大小写
gu 转换为小写
gU 转换为大写

1.2 单词 & 字串

概念 说明
单词 以 非字母 分隔，由字母、数字、下划线或其他非空白字符的序列组成
字串 以 **空白字符 **分隔，由非空白字符序列组成

1.3 文本对象及范围

1 文本对象

文本对象 说明
单词或字串 w/W（word）
句子 s（sentence）
段落 p（paragraph）
引号 '"`（单 双 反）
括号 <>()[]{}

2 范围

范围 说明
i inside
a around

3 组合

命令 说明
iw 当前单词
aw 当前单词及其后空白
iW 当前字串
aW 当前字串及其后空白
is 当前句子
as 当前句子及其后空白
ip 当前段落
ap 当前段落及一个空行
i'"` 引号内部
a'"` 引号及其后空白
i>)]} 括号内部
a>)]} 括号及其内部
it Html 或 Xml 一对标签内部
at Html 或 Xml 一对标签及其内部

二 帮助

命令 说明
vimtutor 练习手册
:help 帮助手册
ctrl-] 跳转到手册中的光标所在链接的详细页面
ctrl-t 返回到原来的位置

三 开关

3.1 打开

命令 说明
vim file1 file2 ... 打开/新建 1 或多个文件
vim -R file 只读模式，但可以 :w! 强制保存
vim -M file 只读模式，不能强制修改
vim -o file1 file2... 为每一个文件打开一个窗口
:edit file 在 vim 中打开另一个文件，可以缩写 :e
:args file1 file2 ... 在 vim 中编辑另一个文件列表

3.2 退出

命令 说明
:q 不保存并退出
:q!，ZQ 不保存并强制退出
:e! 回滚所有修改（把磁盘文件重新读入缓冲区）

3.3 保存

命令 说明
:w 保存（把缓冲区写入磁盘文件）
:w 文件名 另存一个新文件，继续编辑原文件
:saveas 文件名 另存一个新文件，并开始编辑它
:file 文件名 另编辑一个新文件，但未保存
:wq 保存并退出，不管文件有没有被修改，都会修改文件的时间戳，即会触发重新编译
:x，ZZ 保存并退出，若文件没有修改，不会修改文件的时间戳，即不会触发重新编译
:w !sudo tee % 以超级用户权限保存文件
:[range]w 文件名 将指定范围的内容保存到指定文件中
:[range]w >>文件名 将指定范围的内容追加到指定文件中

四 查看

命令 说明
ctrl-g 查看光标所在位置和文件状态信息
ga 查看光标所在字符的进制数

五 光标移动

5.1 上下左右

命令 说明
h j k l 左 下 上 右
gj gk 下 上，表示移动屏幕行（有时一行太长可能会显示为多行）

5.2 行

命令 说明
0 移动到行首
^ 移动到行首的非空白符
$ 移动到行尾
g_ 移动到行内最后一个非空白符
`3 `
gg 移动到文件第一行
G 移动到文件最后一行
3gg，3G 移动到文件第 3 行

5.3 单词 & 字串

命令 说明
w 移动到下一个单词的开头
b 若光标不在当前单词的开头，则移动到当前单词的开头
若光标在当前单词的开头，则移动到上一个单词的开头
e 若光标不在当前单词的结尾，则移动到当前单词的结尾
若光标在当前单词的结尾，则移动到下一个单词的结尾
ge 移动到上一个单词的结尾
W B E gE 小写是面向 单词 的移动，而大写是面向 字串 的的移动

5.4 匹配

命令 说明
% 跳转到配对的括号
fx 移动到字符 x 在当前行下次出现的位置
Fx 移动到字符 x 在当前行上次出现的位置
tx 移动到字符 x 在当前行下次出现的位置的前一个字符
Tx 移动到字符 x 在当前行上次出现的位置的前一个字符
; 重复之前的 f、t、F、T 操作
, 反向重复之前的 f、t、F、T 操作

5.5 句子段落

命令 说明
) 移动光标到句子的末尾
( 移动光标到句子的开头
} 移动光标到当前段落的末尾
{ 移到光标到当前段落的开头

5.6 屏幕

命令 说明
H 移动光标到屏幕的第一行
M 移动光标到屏幕的中间一行
L 移动光标到屏幕的最后一行
zz 移动屏幕使光标居中

5.7 翻页

命令 说明
ctrl-f 下一页
ctrl-b 上一页
ctrl-d 下半页
ctrl-u 上半页
ctrl-e 屏幕向下滚动一行（光标不动）
ctrl-y 屏幕向上滚动一行（光标不动）

六 编辑

6.1 插入

命令 说明
i 在光标前开始插入
I 在行首开始插入
a 在光标后开始插入
A 在行尾开始插入
o 在下一行插入
O 在上一行插入

6.2 删除（剪切）

1 normal mode

命令 说明
x，dl 删除当前字符
X，dh 删除前一个字符
dd 删除当前行
d3d，3dd 删除 3 行
d0 删除光标所在位置到行首的内容
D，d$ 删除光标所在位置到行尾的内容
dw 删除光标所在位置到下一个单词开头（不包含）的内容
diw 删除光标所在单词
daw 删除光标所在单词及其后边的空白
J 删除换行符，将下一行合并到当前行，以空格隔开
gJ 将下一行合并到当前行，之间不含空格

2 insert mode

命令 说明
ctrl-h 同 backspace
ctrl-w 删除从光标位置到单词开头
ctrl-u 删除从光标位置到行开头

6.3 修改

命令 说明
s，cl 删除当前字符，并进入插入模式
S，cc 删除当前行，并进入插入模式（保持原有缩进）
C，c$ 删除光标所在位置到行尾的内容，并进入插入模式
~ 修改光标下字符的大小写，并移动到下一个字符

6.4 撤销

命令 说明
u 撤销（undo）
U 撤销当前行的所有改动
ctrl-r 取消撤销（redo）

6.5 替换

命令 说明
r 替换当前字符
gr 替换当前字符（虚拟替换模式）
R 一直替换直到按下 ESC
gR Virtual Replace Mode（虚拟替换模式）
  • 某些特殊字符在文件中以单个字符表示，但在屏幕上却占据若干列的宽度（如制表符默认宽度是 8）
    • 在一个制表符上使用 R，则输入 1 个字符就会替换制表符，即把屏幕上 8 个字符替换成了 1 个字符，会导致当前行缩短
    • 在一个制表符上使用 gR，则输入前 7 个字符都会插入到制表符之前，输入第 8 个字符才会替换制表符

6.6 复制粘贴

1 normal mode

命令 说明
yy，Y 复制一行
yw 包括单词后面的空白字符
ye 不包括单词后面的空白字符
p 在光标后粘贴
P 在光标前粘贴

2 insert mode

命令 说明

七 EX命令

7.1 概述

  1. 出于历史原因，在 Vim 命令行模式中执行的命令又被称做 Ex 命令
  2. Ex 命令都需要先输入 : 后按回车才能执行
  3. Vim 为几乎所有功能都提供了相应的 Ex 命令，通过命令 :h ex-cmd-index 可获得完整的 Vim Ex 命令列表
  4. 在命令行模式下：% 代表文件的完整路径
  5. @:重复上次 Ex 命令

7.2 Ex 命令操作范围 range

range 说明
. 当前行
$ 最后一行
% 同 1,$，整个文件
m 第 m 行
m,n 从第 m 行到第 n 行
m,+n 从第 m 行到当前行以下 n 行，n 以当前行作为参考
m;+n 从第 m 行到 m 行以下 n 行，n 以 m 作为参考
'<,'> 在可视模式下输入 :，命令行会预先填充此范围，代表高亮选中内容

7.3 目标地址 address

address 说明
. 当前行下方
n 第 n 行下方
0 第 0 行下方，即文件开头
$ 最后一行下方，即文件末尾

7.4 正则表达式

1 概述

  1. Vim 的搜索、替换和 global 等命令均支持正则表达式指定 pattern
  2. .*[]^%/\?~ 等字符有特殊含义需要转义

2 预定义字符

元字符 等价于 说明
. [^\n] 表示匹配任意一个除换行符 ( \n ) 外的其他字符
\l [a-z] 任意一个小写字母
\L [^a-z] 任意一个除小写字母外的字符
\u [A-Z] 任意一个大写字母
\U [^A-Z] 任意一个除大写字母外的字符
\w [0-9a-zA-z_] 数字、字母、下划线
\W [^0-9a-zA-z_] 除了数字、字母和下划线外的任意一个字符
\d [0-9] 任意一个数字
\D [^0-9] 任意一个非数字字符
\x [0-9A-Fa-f] 任意一个十六进制数位
\X [^0-9A-Fa-f] 任意一个非十六进制数位
\s [\f\r\n\t\v] 任意一个空白字符（空格、换行符、制表符和换页符）
\S [^\f\r\n\t\v] 任意一个非空白字符

3 位置

元字符 说明
^ 行首
$ 行尾
\< 单词词首
\> 单词词尾

7.5 搜索

1 说明

  1. 搜索不算是 Ex 命令但类似
  2. 如果搜索模式为空，则使用最近一次的搜索模式
  3. .*[]^%/\?~$ 有特殊含义，需要转义
  4. 单个模式里的大小写
    • pattern 以 \c 开头表示这次搜索模式不区分大小写
    • pattern 以 \C 开头表示这次搜索模式区分大小写
  5. 历史纪录
    1. 输入 / ? 进入搜索模式后，可以用 ↑↓ 来使用搜索模式历史纪录
    2. 还可以输入 /haha 后，使用 ↑↓ 来使用历史纪录中以 haha 开头的

2 命令

命令 说明
/pattern 向下搜索
?pattern 向上搜索
* 向下搜索光标所在单词（完全匹配），相当于加了 \<\>
# 向上搜索光标所在单词（完全匹配）
g* 向下搜索光标所在单词（部分匹配）
g# 向上搜索光标所在单词（部分匹配）
n 搜索后，查找下一个
N 搜索后，查找上一个

7.6 替换

option 说明
g global：全局替换，默认只替换每行第一个匹配
c confirm：每个替换都会询问是否执行
i ignorecase：不区分大小写
p 在命令执行的时候打印出最后一个被修改的行
  • c 待选择操作的含义：
    • y：Yes，执行替换
    • n：No，跳过
    • a：All，对剩下的匹配点全部执行替换，不需要再确认
    • q：Quit，不再执行任何替换
    • l：Last，替换完当前匹配点后退出
