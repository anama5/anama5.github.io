一 概述

1.1 CSS简介

  1. CSS：Cascading Style Sheets 层叠样式表

  2. 负责页面的布局和样式

  3. CSS1 & CSS2 是单一规范，由于内容增多，CSS 划分了多个模块，各个模块可能处于不同的 Level

1.2 语法

二 Html 引入 CSS 的方式

方式 css 书写位置
行内 Html 标签的 style 属性中
内嵌 当前页面的 <style> 标签中
外联 外部 .css 文件中
  • link 和 @import 方式的区别
    1. link 属于 HTML 标签，而 @import 是 CSS 提供的，需要写在 <style> 中
    2. @import 只在 IE 5 以上才能识别，而 link 是 HTML 标签，无兼容问题
    3. 页面被加载时，link 会同时被加载，而 @import 引用的 CSS 会等到页面被加载完再加载
    4. link 方式的样式的权重高于 @import 权重
    5. @import 不支持js的动态修改
    6. 一般推荐link，@import 补充使用

三 URI & URL

3.1 概述

项 说明
URI（Uniform Resource Identifier） 统一资源标志符
URL（Uniform Resource Locator） 统一资源定位符
URN（Uniform Resource Name） 统一资源名称
  1. 能把一个资源唯一的标识出来就是 URI，唯一标识资源的方式有很多，其中就包括 URL 和 URN，所以 URL 和 URN 都属于 URI，可以看作是 URI 的一种实现方式
  2. 例如要标识一个人
    1. URL：地址 + 姓名
    2. URN：身份证号

3.2 URL 格式

项 说明
scheme（protocol） 协议，如 http, https, ftp, mailto, file, data, irc
host 主机，hostname, ip
port 端口
path 资源在主机上的路径
query 请求参数，key=value&key=value&…
fragment（hash） 用来标识 URL 所标识资源里的某个资源
  1. fragment 不会被发送到服务器，fragment 的改变不会触发浏览器刷新页面，但是会生成浏览历史
    1. 这两个特性主要应用是单页面路由，JS 提供了 location.hash 来操作当前 URL 的 fragment，同时提供了 Hashchange 事件来监听 fragment 的改变
    2. 利用 JS 的 API + 这两个特性就可以实现一个简单前端路由
  2. fragment 会被浏览器根据文件媒体类型（MIME type）进行对应的处理
  3. Google 的搜索引擎会忽略 # 及其后面的字符串，可以使用 #! 的格式，Google 的搜索引擎就会将 fragment 转为 query 的格式

四 CSS 文档

4.1 文档符号

1 基本符号

符号 说明
[] 表示组
<> 值类型

2 分割符号

符号 说明
  空格前后的各部分必须出现，且按顺序出现
&& 各部分必须出现，但可以不按顺序
|| 各部分至少出现一个，可以不按顺序
| 各部分只能出现一个
  • 优先级：数量 > “空格” > “&&” > “||” > “|”

3 数量符号

符号 说明
  无数量符号：表示一次
* 任意次数
? 零次或一次
+ 一次或多次
# 一次或多次，多次出现必须以逗号分隔
{m,n} 至少 m 次，至多 n 次
! 用在组后，表示该组至少产生一个值；即使组内项目的语法允许省略全部的值，也至少要保留一个值

4.2 Values and Units

1 Units

1 Distance Unites

相对单位 说明
em 相对于当前元素的字体大小
rem 相对于根元素 <html> 的字体大小
ex 相对于小写字母 x 的高度即 x-height
ch 相对于字符数字 0 的宽度
vw 相对于视窗的宽度：视窗宽度是100 vw
vh 相对于视窗的高度：视窗高度是100 vh
vmin 等于 vh 和 vw 中较小的值
vmax 等于 vh 和 vw 中较大的值
绝对单位 说明
px 像素
cm 厘米
mm 毫米
in 英寸

2 Angle Unites

单位 说明
deg 角度，一个完整圆角度是 360
grad 百分度，一个完整圆百分度是 400
rad 弧度，一个完整圆弧度是 2π
turn 圈数，一个完整圆圈数是 1

3 Duration Unites

单位 说明
s 秒
ms 毫秒

4 Frequency Units

单位 说明
Hz 赫兹
kHz 千赫兹

5 Resolution Units

单位 说明
dpi 表示每英寸点数
dpcm 表示每厘米点数
dppx 表示每像素点数
x dppx 的别名

1 Textual Data Types

类型 取值
<string> '' 或 ""

2 Numeric Data Types

类型 取值
<integer> 整数
<number> 实数
<percentage> <number> + %，通常为相对于父级元素的大小
<ratio> <number [0,∞]> [ / <number [0,∞]> ]?，两个非负数的比值，第二个缺省值为 1

<resolution> <number> + 分辨率单位
