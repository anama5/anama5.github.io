05-样式

发表于 2022-02-28 分类于 CS ， CSS

一 列表

1.1 list-style

1.2 list-style-type

值 说明
disc（默认） 实心圆点
none 无样式
circle 空心圆点
square 实心方块
decimal 从1开始的数字
...... 还有许多样式

1.3 list-style-image

值 说明
none（默认） 使用 list-style-type 的样式
<url> url()，使用指定的图标

1.4 list-style-position

值 说明
outside（默认） 标记盒 ::marker 在主块盒之外（即 list-style-type 在外部）
inside 标记盒 ::marker 在主块盒之内，是主块盒中的第一个行内盒

二 表格

2.1 border-collapse

值 说明
separate（默认） 分离，HTML 表格的传统模式，相邻单元格都拥有独立的边框
collapse 合并，表格中相邻单元格共享边框

2.2 border-spacing

2.3 empty-cells

值 说明
show（默认） 空单元格的背景和边框正常显示
hide 空单元格的背景和边框隐藏

2.4 caption-side

值 说明
top（默认） 标题在表格顶部，标题是在表格外面的
bottom 标题在表格底部，标题是在表格外面的

2.5 table-layout

值 说明
auto（默认） 表格及单元格的宽度取决于其包含的内容
fixed 表格和列的宽度通过表格的宽度来设置，某一列的宽度仅由该列首行的单元格决定

三 溢出

值 说明
visible（默认） 溢出内容可见
hidden 溢出内容隐藏
scroll 无论是否溢出都显示滚动条
auto 如果内容溢出，就显示滚动条，反之则不显示

四 可见性：visibility

值 说明
visible（默认） 元素正常显示
hidden 隐藏元素，但是其他元素的布局不改变，相当于此元素变成透明。若其子元素设为visible，则子元素依然可见
collapse 用于 <table> 行、列、列组和行组，隐藏表格的行或列，并且不占用任何空间，就好像折叠了单元格一般（效果同 display:none ）
  1. display: none：该元素以及它的所有后代元素都会隐藏，不保留位置（可以理解为没了），不影响计数器的值
  2. visibility: hidden：如果后代元素 visibility: visible，则后代元素依然可见，保留位置（可以理解为隐身），会影响计数器的值

五 透明度

值 说明
1（默认） 完全不透明
0 完全透明

六 用户界面样式

6.1 cursor（鼠标样式）

  • 定义鼠标悬浮在元素上方时的样式
值 说明
auto（默认） 浏览器根据内容类别自动选择光标形状
default 箭头
pointer 小手
move 移动
text 文本
not-allowed 禁止
none 隐藏光标，比如看视频时
help 帮助
... 还有很多样式

6.2 outline（轮廓线）

1 语法

  1. 轮廓线不占据空间
  2. 通常不使用轮廓线 outline: none;

2 outline-width

值 说明
<legnth> 指定值
thin 薄
medium 中等
thick 厚

3 outline-style

值 说明
none 无轮廓线
solid 单实线
double 双实线
dotted 点画线
dashed 虚线
groove 雕刻效果，与ridge相反
ridge 浮雕效果，与groove相反
inset 陷入效果
outset 突出效果

6.3 resize

值 说明
none 不允许用户缩放元素
both 允许用户在水平和垂直方向上调整元素的大小
horizontal 允许用户在水平方向上调整元素的大小
vertical 允许用户在垂直方向上调整元素的大小
