一 前言

1.1 HTML 引入 CSS

1 分类

分类 css 书写位置
行内样式 Html 标签的 style 属性中
内部样式 当前页面的 <style> 标签中
外部样式 外部 .css 文件中

2 外部样式的引入

  1. <link> 属于 HTML 标签，除了引入 CSS 外还能引入其它；而 @import 是 CSS 提供的，只能引入CSS，且需要写在 <style> 中
  2. @import 只在 IE 5 以上才能识别，而 <link> 是 HTML 标签，无兼容问题
  3. 页面被加载时，<link> 会同时被加载，而 @import 引用的 CSS 会等到页面被加载完再加载
  4. <link> 方式的样式的权重高于 @import 权重
  5. @import 不支持 js 的动态修改
  6. 一般推荐<link>，@import 补充使用

1.2 URI & URL

1 概述

项 说明
URI（Uniform Resource Identifier） 统一资源标志符
URL（Uniform Resource Locator） 统一资源定位符
URN（Uniform Resource Name） 统一资源名称
  1. 能把一个资源唯一的标识出来就是 URI，唯一标识资源的方式有很多，其中就包括 URL 和 URN，所以 URL 和 URN 都属于 URI，可以看作是 URI 的一种实现方式
  2. 例如要标识一个人
    1. URL：地址 + 姓名
    2. URN：身份证号

2 URL 格式

项 说明
scheme（protocol） 协议，如 http, https, ftp, mailto, file, data, irc
host 主机，hostname, ip
port 端口
path 资源在主机上的路径
query 请求参数，key=value&key=value&…
fragment（hash） 用来标识 URL 所标识资源里的某个资源
  1. fragment 不会被发送到服务器，fragment 的改变不会触发浏览器刷新页面，但是会生成浏览历史
    1. 这两个特性主要应用是单页面路由，JS 提供了 location.hash 来操作当前 URL 的 fragment，同时提供了 Hashchange 事件来监听 fragment 的改变
    2. 利用 JS 的 API + 这两个特性就可以实现一个简单前端路由
  2. fragment 会被浏览器根据文件媒体类型（MIME type）进行对应的处理
  3. Google 的搜索引擎会忽略 # 及其后面的字符串，可以使用 #! 的格式，Google 的搜索引擎就会将 fragment 转为 query 的格式

三 选择器

3.1 基础选择器

pattern represents
S₁, S₂, S₃, ... 并集
S₁S₂S₃... 交集
* any element
E an element of type E
E#id an E element with ID equal to “id”
#id an element with ID equal to “id”，等价于 *#id
E.class an E element with class~=”class”
.class all elements with class~=”class”，等价于 *.class

3.2 关系选择器

pattern 说明
A E E 为 A 的后代元素
A > E E 为 A 的子元素
A ~ E E 为 A 的兄弟元素
A + E E 为 A 的下一个兄弟元素
  • 注：浏览器是从右往左读的，所以关系选择器效率不如基础选择器

3.3 属性选择器

pattern 说明
[attr] 带有 attr 属性的元素
[attr=val] 带有 attr 属性且值为 val 的元素
[attr^=val] 带有 attr 属性且值以 val 开头的元素
[attr$=val] 带有 attr 属性且值以 val 结尾的元素
[attr*=val] 带有 attr 属性且值包含 val 的元素
`[attr =val]`
[att~=val] 带有 attr 属性且值为空白分割的单词列表（其中一个是 val）

3.4 伪类选择器

1 动态伪类

pattern 说明
:link 未访问（有 href 属性的 a 元素）
:visited 已访问
:hover 悬停
:active 激活
:focus 获取焦点

  1. 建议顺序 L V F H A

  2. hover/active 除了 a 外，还能用于其它元素

  3. 直接给 a 标签设置样式，相当于给 a 元素的所有动态伪类都设置了

  4. 如果不想让 a 标签被 tab 键选中

2 结构伪类

1 根

pattern 说明
:root <html> 元素

2 无类型

pattern 说明
:first-child 第一个子元素
:last-child 最后一个子元素
:noly-child 唯一子元素
:nth-child(An + B) 从前向后匹配符合表达式结果的子元素
:nth-last-child(An + B) 从后向前匹配符合表达式结果的子元素
  1. 子元素从 1 开始编号
  2. 通常使用交集的写法
    1. :firs-child：匹配父元素下的第一个子元素
    2. E:first-child：交集，匹配父元素下的第一个且是 E 类型的子元素。若不是第一个或第一个不是 E 类型则无法匹配

3 带类型

pattern 说明
:fitst-of-type 匹配父元素下每种类型中的第一个子元素
:last-of-type 匹配父元素下每种类型中的最后一个子元素
:noly-of-type 匹配父元素下每种类型唯一的子元素
:nth-of-type(An + B) 从前向后匹配每种类型中符合表达式结果的子元素
:nth-last-of-type(An + B) 从后向前匹配每种类型中符合表达式结果的子元素
  1. :first-of-type：匹配父元素下以类型为单位，每种类型中的第一个子元素，不要求一定是父元素的第一个子元素
  2. E:first-of-type：交集，匹配父元素下 E 类型中的第一个子元素

4 An + B

表达式示例 说明
2n，even 偶数
2n+1，2n-1，odd 奇数
-n+m 前 m 个
n+m 从第 m 个往后

3 元素状态伪类

pattern 说明
:checked 选中的 radio checkbox option
:default 默认选中的 radio checkbox option
:disabled 禁用的元素
:enabled 启用的元素
:empty 没有子元素的元素

4 否定伪类

pattern 说明
:not(S) 非（权重由括号内选择器决定，自身0，不支持复杂选择器，可嵌套）

5 目标伪类

pattern 说明
:target 选中当前被激活的锚点

3.5 伪元素选择器

pattern 说明
::before 在父元素内容的最前添加一个伪元素
::after 在父元素内容的最后添加一个伪元素
::first-letter 匹配块级元素第一行的第一个字母
::first-line 匹配块级元素第一行
::selection 匹配被用户选中或者处于高亮状态的部分
::placeholder 匹配 placeholder 文本
  1. 伪元素：一般页面中非主体内容可以使用伪元素
    • 元素：HTML 设置的标签
    • 伪元素：由 CSS 模拟出的标签效果
  2. ::before, ::after 必须设置 content 属性才生效
  3. 伪元素默认是行内元素

四 Values and Units

4.1 语法符号

1 组合符号

符号 说明
  空格前后的各部分必须按照给定的顺序全部出现
&& 各部分必须出现，但可以不按顺序
|| 各部分至少出现一个，可以不按顺序
| 各部分只能出现一个
  • 优先级：数量 > “空格” > “&&” > “||” > “|”

2 数量符号

符号 说明
  无数量符号：表示一次
* 任意次数
? 零次或一次，即可选
+ 一次或多次
# 一次或多次，多次出现必须以逗号分隔
{m,n} 至少 m 次，至多 n 次
! 用在组后，表示该组至少产生一个值；即使组内项目的语法允许省略全部的值，也至少要保留一个值

3 其它符号

符号 说明
[] 表示组
<> 数据类型
, 无特殊意义，用来分割多个并列值
/ 无特殊意义，用来分割一个值的多个部分，多用在缩写属性中，/ 前后的数据类型通常相同但属于不同的属性

4.2 Units

1 Distance Unites

相对单位 说明
em 相对于当前元素的字体大小
rem 相对于根元素 <html> 的字体大小
ex 相对于小写字母 x 的高度即 x-height
ch 相对于字符数字 0 的宽度
vw 相对于视窗的宽度：视窗宽度是100 vw
vh 相对于视窗的高度：视窗高度是100 vh
vmin 等于 vh 和 vw 中较小的值
vmax 等于 vh 和 vw 中较大的值
绝对单位 说明
px 像素
cm 厘米
mm 毫米
in 英寸

2 Angle Unites

单位 说明
deg 角度，一个完整圆角度是 360
grad 百分度，一个完整圆百分度是 400
rad 弧度，一个完整圆弧度是 2π
turn 圈数，一个完整圆圈数是 1

3 Duration Unites

单位 说明
s 秒
ms 毫秒

4 Frequency Units

单位 说明
Hz 赫兹
kHz 千赫兹

5 Resolution Units

单位 说明
dpi 表示每英寸点数
dpcm 表示每厘米点数
dppx 表示每像素点数
x dppx 的别名

类型 取值
