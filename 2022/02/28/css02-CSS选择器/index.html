02-CSS选择器

发表于 2022-02-28 分类于 CS ， CSS

一 权重

规则 粒度
!important ∞
行内样式 1, 0, 0, 0
id 0, 1, 0, 0
class，伪类，属性 0, 0, 1, 0
标签，伪元素 0, 0, 0, 1
* 0, 0, 0, 0
继承 null（继承没有权重，比 0 小）

  1. 权重的四个数有低位高位的概念

    1. 不会进位，如 0,0,0,8 + 0,0,0,7 = 0,0,0,15
    2. 从左到右依次代表：行内样式的个数、id选择器的个数、类选择器的个数、标签选择器的个数

  2. 权重的计算

    1. 判断选择器能否直接选中标签，不能则说明是继承，优先级最低
      • 全部是继承的时候，计算其父元素的权重
      • 一直重复，直到结果
    2. 计算所有选择器的权重之和，然后从左到右比较每一位的大小，有结果则不再继续
    3. 如果优先级相同，谁在后用谁的

二 基础选择器

选择器 说明
html 标签选择器
#id id 选择器
.class class 选择器
* 通配选择器
S₁, S₂, S₃, ... 并集
S₁S₂S₃... 交集

三 关系选择器

关系选择器 说明
A E E 为 A 的后代元素
A > E E 为 A 的子元素
A ~ E E 为 A 的兄弟元素
A + E E 为 A 的下一个兄弟元素
  • 注：浏览器是从右往左读的，所以关系选择器效率不如基础选择器

三 属性选择器

属性选择器 说明
[attr] 带有 attr 属性的元素
[attr=value] 带有 attr 属性且值为 value 的元素
[attr^=value] 带有 attr 属性且值以 value 开头的元素
[attr$=value] 带有 attr 属性且值以 value 结尾的元素
[attr*=value] 带有 attr 属性且值包含 value 的元素

四 伪类选择器

4.1 动态伪类

选择器 说明
:link 未访问
:visited 已访问
:hover 悬停
:active 点击
:focus 获取焦点的元素

  1. 建议顺序 L V F H A

  2. hover/active 除了 a 外，还能用于其它元素

  3. 直接给 a 标签设置样式，相当于给 a 元素的所有动态伪类都设置了

  4. 如果不想让 a 标签被 tab 键选中

4.2 结构伪类

1 无类型

选择器 说明
:first-child 第一个子元素
:last-child 最后一个子元素
:noly-child 唯一子元素
:nth-child(表达式) 从前向后匹配符合表达式结果的子元素
:nth-last-child(表达式) 从后向前匹配符合表达式结果的子元素
  1. 子元素从 1 开始编号
  2. 通常使用交集的写法
    1. :firs-child：匹配父元素下的第一个子元素
    2. E:first-child：交集，匹配父元素下的第一个且是 E 类型的子元素。若不是第一个或第一个不是 E 类型则无法匹配

2 带类型

选择器 说明
:fitst-of-type 匹配父元素下每种类型中的第一个子元素
:last-of-type 匹配父元素下每种类型中的最后一个子元素
:noly-of-type 匹配父元素下每种类型唯一的子元素
:nth-of-type(表达式) 从前向后匹配每种类型中符合表达式结果的子元素
:nth-last-of-type(表达式) 从后向前匹配每种类型中符合表达式结果的子元素
  1. :first-of-type：匹配父元素下以类型为单位，每种类型中的第一个子元素，不要求一定是父元素的第一个子元素
  2. E:first-of-type：交集，匹配父元素下 E 类型中的第一个子元素

3 表达式

表达式示例 说明
2n，even 偶数
2n+1，2n-1，odd 奇数
-n+m 前 m 个
n+m 从第 m 个往后

4.3 元素状态伪类

选择器 说明
:checked 选中的 radio checkbox option
:default 默认选中的 radio checkbox option
:disabled 禁用的元素
:enabled 启用的元素
:empty 没有子元素的元素
:root <html> 元素

4.4 否定伪类

选择器 说明
:not(S) 非（权重由括号内选择器决定，自身0，不支持复杂选择器）

4.5 目标伪类

选择器 说明
:target 选中当前被激活的锚点

五 伪元素选择器

伪元素选择器 说明
::before 在父元素内容的最前添加一个伪元素
::after 在父元素内容的最后添加一个伪元素
::first-letter 匹配块级元素第一行的第一个字母
::first-line 匹配块级元素第一行
::selection 匹配被用户选中或者处于高亮状态的部分
  1. 伪元素：一般页面中非主体内容可以使用伪元素
    • 元素：HTML 设置的标签
    • 伪元素：由 CSS 模拟出的标签效果
  2. ::before, ::after 必须设置 content 属性才生效
  3. 伪元素默认是行内元素
