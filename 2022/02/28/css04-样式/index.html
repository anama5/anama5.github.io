一 图片

1.1 图片类型

  1. 位图
    1. 来自相机、扫描仪或由软件生成
    2. 由像素组成
    3. 格式：jpg、png、gif、webp
    4. 放大会失真
  2. 矢量图
    1. 来自数学工具软件（如 Illustrator）
    2. 由数学矢量（直线、曲线…）构成
    3. 格式：svg
    4. 缩放不失真，可用于图标、logo、简单图形或背景

1.2 位图格式

格式 colors（色深） transparency（透明度） animation（动画）
JPEG 16M
GIF 256 ⭐ ⭐
PNG 16M ⭐ ⭐
Webp 16M ⭐ ⭐

1.3 Webp

  1. 如何转换成 webp

    1. Photoshop（需要 webpshop 插件）
    2. Sketch
    3. 在线工具
    4. 命令行工具

  2. 使用

    1. 浏览器支持：直接用 <img>

    2. 浏览器不支持

二 background

2.1 background-color

项 说明
Initial transparent
Applies to all elements
Inherited no
值 说明
transparent（默认） 透明
currentColor 当前元素的字体颜色
<color> 设置颜色

2.2 background-image

项 说明
Initial none
Applies to all elements
Inherited no
值 说明
none（默认） 无背景图
url url(地址不提倡加引号)
<gradient> 渐变

2.3 background-repeat

项 说明
Initial repeat
Applies to all elements
Inherited no
单值 双值 说明
repeat（默认） repeat repeat 平铺
repeat-x repeat no-repeat x 轴平铺
repeat-y no-repeat repeat y 轴平铺
no-repeat no-repeat no-repeat 不平铺
space space space 两端对齐平铺，多出来的空间用空白代替
round round round 两端对齐平铺，多出来空间通过自身的拉伸来填充

2.4 background-position

项 说明
Initial 0% 0%
Applies to all elements
Inherited no
Percentages 相对于 background-size 的 width/height

2.5 background-size

项 说明
Initial auto auto
Applies to all elements
Inherited no
Percentages relative to the background positioning area
值 说明
auto（默认） 以背景图片的本身大小显示
cover 等比例缩放背景图片以完全覆盖背景区，背景区没有空白，背景图片可能部分看不见
contain 等比例缩放背景图片以完全装入背景区，背景区可能部分空白
<length> 长度
<percentage> 背景图片相对当前盒子宽高的百分比

2.6 background-attachment

决定背景图像的位置是在视口内固定，还是随着包含它的区块滚动

项 说明
Initial scroll
Applies to all elements
Inherited no
值 说明 元素滚动条 页面滚动条
scroll（默认） 背景相对于元素本身固定，融为一体 背景不动 背景会动
fixed 背景相对于视口固定 背景不动 背景不动
local 背景相对于元素的内容固定 背景会动 背景会动

2.7 background-origin

决定背景图片相对于哪个盒子定位

项 说明
Initial padding-box
Applies to all elements
Inherited no

2.8 background-clip

设置元素的背景切割范围，相对于哪个盒子定位不变

项 说明
Initial border-box
Applies to all elements
Inherited no
值 说明
border-box（默认） 背景延伸至边框外沿（但是在边框下层）
content-box 背景被裁剪至内容区外沿
padding-box 背景延伸至内边距外沿

四 表格

4.1 border-collapse

项 说明
Initial separate
Applies to table and inline-table elements
Inherited yes
值 说明
separate（默认） 分离，HTML 表格的传统模式，相邻单元格都拥有独立的边框
collapse 合并，表格中相邻单元格共享边框

4.2 border-spacing

设置单元格之间的间距，border-collapse: separate 才有效

项 说明
Initial 0
Applies to table and inline-table elements
Inherited yes

4.3 empty-cells

项 说明
Initial show
Applies to table-cell elements
Inherited yes
值 说明
show（默认） 空单元格的背景和边框正常显示
hide 空单元格的背景和边框隐藏

4.4 caption-side

设置标题的位置，注：标题是在表格外面的

项 说明
Initial top
Applies to table-caption elements
Inherited yes

4.5 table-layout

项 说明
Initial auto
Applies to table and inline-table elements
Inherited no
值 说明
auto（默认） 表格及单元格的宽度取决于其包含的内容
fixed 表格和列的宽度通过表格的宽度来设置，某一列的宽度仅由该列首行的单元格决定

五 Visual effects

5.1 overflow

项 说明
Initial visible
Applies to block containers
Inherited no
值 说明
visible（默认） 溢出内容可见
hidden 溢出内容隐藏
scroll 无论是否溢出都显示滚动条
auto 如果内容溢出，就显示滚动条，反之则不显示

5.1 visibility

项 说明
Initial visible
Applies to all elements
Inherited yes
值 说明
visible（默认） 元素正常显示
hidden 隐藏元素，但是其他元素的布局不改变，相当于此元素变成透明。若其子元素设为visible，则子元素依然可见
collapse 用于 <table> 行、列、列组和行组，隐藏表格的行或列，并且不占用任何空间，就好像折叠了单元格一般（效果同 display:none ）

5.3 opacity

让元素整体（包括内容）一起变透明

项 说明
Initial visible
Applies to all elements
Inherited no
Percentages map to the range [0,1]
值 说明
1（默认） 完全不透明
0 完全透明

6.3 resize

specify whether or not an element is resizable by the user

项 说明
Initial none
Applies to elements with overflow other than visible, and optionally replaced elements such as images, videos, and iframes
Inherited no
值 说明
none 不允许用户缩放元素
both 允许用户在水平和垂直方向上调整元素的大小
horizontal 允许用户在水平方向上调整元素的大小
vertical 允许用户在垂直方向上调整元素的大小

6.4 filter（滤镜）

filter 说明
blur() 模糊，取值 radius 设定高斯函数标准差，或 <number> + px，默认 0
brightness() 亮度，取值 0% ~ 100%（可超，会更亮），默认 1
contrast() 对比度，取值 0% ~ 100%（可超），默认 1
grayscale() 灰度，取值 0% ~ 100%，默认 0
saturate() 饱和度，取值 0% ~ 100%（可超），默认 1

6.5 clip-path

  1. clip-path：裁剪出元素的部分区域做展示，区域内的部分显示，区域外的隐藏
  2. 即使做了裁剪，容器的占位大小是不变的
